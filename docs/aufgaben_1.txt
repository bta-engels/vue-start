Vue Aufgaben

1. Erstellt im Verzeichnis 'components' eine Komponente 'AddTodo' mit einem
Formular zum Erstellen eines neuen 'Todo' Eintrags. Felder:
- text: title
- button 'add Todo'

2. Erstellt im Verzeichnis 'components' eine Komponente 'Todo' für die Anzeige eines
einzelnen Todo Eintags als Listen Elements mit einem Formular:
- checkbox: name 'done'
- text: name 'text'
- button zum löschen eines todo's

3. benutzt beide Komponenten in der Todos Komponente, statt der dort hart codierten
HTML Anweisungen.
